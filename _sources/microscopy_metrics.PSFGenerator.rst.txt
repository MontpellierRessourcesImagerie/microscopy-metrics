microscopy_metrics.scripts.PSFGenerator package
===============================================

This package contains the PSFGenerator script used for generating point spread functions (PSFs) for microscopy data. The PSFGenerator script has its own unique algorithm for generating PSFs based on the imaging modality. It also provides methods for generating aberrated PSFs based on the imaging modality and the aberration parameters.
