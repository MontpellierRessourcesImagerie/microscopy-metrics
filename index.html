Microscopy Metrics Documentation

Microscopy Metrics is a Python package designed for metrology of microscopy systems using Point Spread Function (PSF) analysis. This tool enables you to characterize optical aberrations, evaluate system resolution, and generate detailed reports for quality control in microscopy.

Whether you’re a researcher, engineer, or developer, Microscopy Metrics provides the tools you need to analyze, visualize, and optimize your microscopy data.

—

Note

This project is under active development. The current version is a work in progress and may not be fully stable. We encourage users to report issues or provide feedback to help improve the project. For the latest updates, check out our GitHub repository.
